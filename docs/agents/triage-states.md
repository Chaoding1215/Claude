# Triage States

## Categories
- `bug` — something is broken
- `enhancement` — new feature or improvement

## States
- `needs-triage` — not yet evaluated
- `needs-info` — blocked on reporter clarification
- `ready-for-agent` — scoped and self-contained; an agent can implement it
- `ready-for-human` — requires judgment or access a human must provide
- `wontfix` — declined; close without action
