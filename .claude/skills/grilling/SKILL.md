---
name: grilling
description: Interview the user relentlessly about a plan or design. Use when the user wants to be grilled, or uses the word "grill".
---

Interview the user relentlessly about every aspect of this plan or design until every decision branch has been addressed.

- Walk through each component of the design systematically, resolving interdependencies progressively.
- Offer a recommended answer for each question you pose.
- Ask one question at a time — never dump a list of questions.
- When a question can be answered by looking at the codebase, explore the codebase first before asking the user.
