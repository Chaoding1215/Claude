# LR-0006: Competitive Landscape Analysis (Huawei slide)

**Status**: active

## Source
Huawei internal competitive matrix — comparing OCD, Palo Alto, Fortinet, Huawei across full security portfolio.

## Key findings
- OCD wins on E2E service delivery (integrator model), not product features
- Palo Alto and Fortinet lead in core security products; both have gaps in DevSecOps, managed services
- Huawei has significant gaps: SASE security components, data security, DevSecOps, cloud native, MDR
- For SME/B2B, MDR is the recommended play: low barrier, high value, growing market

## Three lessons built from this analysis
- Lesson 6: SASE (SD-WAN + SSE; Zscaler/Palo Alto/Fortinet competitive dynamics)
- Lesson 7: DevSecOps (CI/CD pipeline security; shift-left; gap for network vendors)
- Lesson 8: MDR (managed detection and response; MSSP vs. MDR; OCD integrator model)
