# LR-0002: Threat Landscape

**Status**: active

## What was covered

- Five threat actor categories: nation-states, organized crime, hacktivists, insider threats, script kiddies
- MITRE ATT&CK: tactics vs. techniques, named groups
- Cyber Kill Chain (7 stages): Recon → Weaponize → Deliver → Exploit → Install → C2 → Act
- Core vocabulary: APT, TTPs, IOC, zero-day, RaaS, threat intelligence

- After that: deep dives per domain tied to how vendors solve the problem.
- Start building a reference glossary — vocabulary is accumulating fast.
