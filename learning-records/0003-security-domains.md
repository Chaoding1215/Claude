# LR-0003: Security Domains

**Status**: active

## What was covered

Six security domains with key technologies, vendors, and problems solved:
- Endpoint (AV → EDR → XDR)
- Network (NGFW, IDS/IPS, NDR, DDoS, DNS security)
- Cloud (CSPM, CWPP, CASB, CNAPP)
- Identity (IAM, MFA, PAM, SSO, Zero Trust)
- Application (SAST, DAST, WAF, API security, SCA)
- SOC/SIEM/SOAR (MTTD, MTTR, threat hunting)

Also: platform vs. best-of-breed debate.
